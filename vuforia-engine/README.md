# Vuforia engine

<figure><img src="../.gitbook/assets/vuforia_logo.png" alt=""><figcaption></figcaption></figure>

Vuforia is one of the most widely used libraries for developing AR applications, not only for **Android** and **iOS** mobile devices but also for smartglasses such as Microsoft HoloLens and Magic Leap.

{% embed url="https://developer.vuforia.com/home" %}

By leveraging the device’s camera and sensors, Vuforia can perform real-time tracking of the user’s or device’s position and orientation, enabling accurate alignment of virtual content with the user's perspective.

### Features Offered

* Image Tracking
* Cylinder Tracking
* 3D Object Tracking
* Surface Detection
* Area Targets
* Multi-target Tracking
* Model Targets (CAD-based recognition)
* VuMarks (custom markers)
* Ground Plane detection
